# 进度 (Progress)

## set()
设置进度条的进度。
- 类型
```ts
set: (value: number) => void
```

## get()
获取进度条的进度。
- 类型
```ts
get: () => number
```

## add()
增加进度条的进度。
- 类型
```ts
add: (value: number) => void
```

## sub()
减少进度条的进度。
- 类型
```ts
sub: (value: number) => void
```

## reset()
重置进度条的进度。
- 类型
```ts
reset: () => void
```

## hide()
隐藏进度条，并重置处理状态。
- 类型
```ts
hide: () => void
```
